Vehicle Registry System 🚗✈️

A complete Vehicle Registry System project developed in C++ using Object Oriented Programming (OOP) concepts.
This project demonstrates:

Class & Object
Encapsulation
Constructors & Destructor
Static Member
Getters & Setters
Array of Objects
All Types of Inheritance:
Single Inheritance
Multilevel Inheritance
Multiple Inheritance
Hierarchical Inheritance
Polymorphism
Dynamic Memory Allocation

📌 Project Overview

The Vehicle Registry System allows users to:

✅ Add Different Types of Vehicles
✅ Display All Registered Vehicles
✅ Search Vehicle By ID
✅ Manage Vehicle Data Using OOP Concepts

This project was created according to the requirements provided in the practical/project PDF.

🛠 Technologies Used

Language: C++
Compiler: GNU G++
Concepts:
OOP
Inheritance
Encapsulation
Virtual Functions
Dynamic Memory Allocation

📚 Inheritance Used

Class	Inheritance Type
Car → Vehicle	Single Inheritance
ElectricCar → Car	Multilevel Inheritance
SportsCar → ElectricCar	Multilevel Inheritance
FlyingCar → Car + Aircraft	Multiple Inheritance
Sedan → Car	Hierarchical Inheritance
SUV → Car	Hierarchical Inheritance

📂 Features

➕ Add Vehicle

Users can add different vehicle types with complete details.

📋 Display All Vehicles

Shows all registered vehicles with formatted output.

🔍 Search Vehicle By ID

Finds a vehicle using its unique vehicle ID.

❌ Exit System

Safely exits the application.

🧠 OOP Concepts Implemented

🔒 Encapsulation

Private data members with public methods.

🏗 Constructor

Used for initializing object values.

🗑 Destructor

Used for memory cleanup.

📌 Static Member

Tracks total vehicle objects.

🔄 Polymorphism

Implemented using virtual functions.
